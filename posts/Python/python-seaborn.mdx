## 1. Overview

Based on matplotlib, Python Seaborn is a Python data structure library for data visualization that works well with pandas. Visualization is the main component of Seaborn that assists in data investigation and interpretation.


## 2. Scope of the tutorial

This tutorial will cover



* Definition of Seaborn in python
* Different ways to plot graphs utilizing Python seaborn.


## 3. The Definition of Seaborn in python.

Utilizing Python Seaborn, the ability to produce and visualize data in the form of graphs, charts, and plots is being improved. Moreover, it shows the data on a statistical graph and any hidden correlations between variables and trends. Seaborn has a top-level interface when compared to Matplotlib. It is a Matplotlib extension module.


## 4. Features of Python Seaborn



* The ability to visualize 3D data
* By employing time series data, Seaborn enables things simple to create graphs.
* Several styles of graphs can be plotted.


## 5. The benefits of using Python Seaborn



* A plot or graph can be used to display the data.
* Execute the data or dataset within the "relplot()" function, and it will arrange the values accurately and calculate automatically, so you don't have to worry about the internals.
* It provides an instructive and attractive display for the loaded dataset.
* You can also visualize categorical data.


## 6. Data visualization utilizing Seaborn

There are two elements in data visualization:



* Visualizing statistical relationships
* Plotting of categorical data

**Visualizing statistical relationships**

It is the process of analyzing relations between different dataset components and understanding how it influences other parameters within the dataset.

### **Boxplot()**

Box-and-whisker plots are the alternative names for box plots. To make it simple to compare one variable with another or various phases of categorical data, it visualizes the distribution of statistical data. The box shows the quartiles of the dataset, and the whiskers stretch to represent the remainder of the distribution, except for points that are designated as "outliers" using a method that is an expression of an interquartile range.


```
sns.boxplot(y='age',data=df)
```

### **Violin plot()**

The same goal is achieved by both a violin plot as well as a whisker and box plot. It shows the distributions of quantitative data for one (or even more) categorical parameters over several phases to enable comparison. In contrast to a box plot, where every single piece of the plot correlates to actual data points, the violin plot incorporates an estimated kernel frequency of the underlying distribution. This method of presenting multiple data distributions simultaneously can be informative and appealing, but it should be noted that the estimated algorithm is influenced by the sample size, making small samples of violins look deceptively smooth.

```
sns.set(style="tricks", color_codes=True)
```

### **relplot()**

Relplot is used to visually represent the connection between two statistically comparable techniques. One could pass data, x-axis and y-axis labels, size, style, color, and many other parameters in this.

```
hue="your_hue", style="your_style", size="and size",)
```

How to plot multiple axes in seaborn?


```
fig, ax= plt.subplots(2,2,figsize=(12,8),dpi=100)

ax[0][0].set_ylabel("y_axis")
```
