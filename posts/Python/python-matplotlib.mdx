## 1. Overview

The method of transforming numbers, text, or massive data sets into different forms of graphs is known as data visualization. The relationship between the data as well as the viewer is depicted in the graph. Matplotlib in python is one of the data visualization techniques.


## 2. Scope of the tutorial

This tutorial covers



* Definition of Python Matplotlib
* Visualizing different methods of plots in matplotlib


## 3. Definition of Matplotlib

A package called Matplotlib is used in Python to display data as graphs, charts, and plots. It displays information in a two-dimensional method. It is mostly used in industrial businesses for statistical analysis.

There are two methods to plot something with Matplotlib.



* Functional Approach
* Object-Oriented Approach
1. **Functional Approach**
* It is the beginner's approach.
* It is used to quickly plot a graph

How to assign the names to the axes?



* xlabel()
* ylabel()

How to provide a title to the plot?

plt.title(“Title Name”)



2. **Object-Oriented Approach**
* This approach is used when we plot multiple plots in the same output area.
* It is also known as the figure and axes approach.

How to create a figure?


```
plt.figure()
```


How to add the axes to the figure?

Syntax


```
fig.add_axes([left_boundary,bottom_boundary,width, height])
```


How to increase the size of the figure?


```
fig,ax=plt.subplots(1,2,figsize=(12,6))
```


How to increase the pixels?


```
dpi - density per inches
fig, ax=plt.subplots(1,2,figsize=(6,4),dpi=100)
```


How are the colors, linestyles, and line widths changed?


```
ax.plot(arr_1,arr_2,color='red',ls='-.',lw=4)
```


Colors available for mapping are - **_rgbcmykw_**

Linestyles

‘-‘- Solid

‘:’- Dotted

‘--‘- Dashed

‘-.’- Dashed-dotted

How to save a figure?

plt.savefig('fig_1.jpeg')

How to show a legend?

plt.legend(loc=1)


## 4. A few of the visualization methods using Matplotlib are:



* Pie plot
* Bar Graph
* Line plot and subplots
* Scatter plot
* Histogram plot
* Area plot


### **Pie plot**

A pie plot displays the statistical data in some kind of circular graph that is divided into defined pieces. A slice is a name given to each division. Businesses and the media can use this kind of plot.

Example


```
# Data labels, sizes, and colors are defined:
labels = 'Strawberry', 'Vanilla', 'Mango', 'Pista'
sizes = [330, 210, 30, 245]
colors = ['red', 'blue', 'green', 'orange']
 
# Data is plotted:
plt.pie(sizes, labels=labels, colors=colors, startangle=90, shadow=True, explode=(0,0.1,0,0), autopct='%1.1f%%')
 
plt.axis('equal')
plt.title("Pie Plot", fontsize = 25)
plt.show()
```

### **Bar graph**

When a rectangular block-shaped categorical value within a single dataset needs to be distinguished, the bar graph is frequently used. Depending on the data, the bar graph might be used to display either horizontally or vertically. Using plt.bar(), a bar graph is drawn in Matplotlib.

Example


```
#Create a line 2D instance with x and y data in sequence xdata, y data
xdata = ['U', 'V', 'W']
ydata = [10, 23, 35]
 
plt.bar(range(len(xdata)),ydata)
plt.title("Bar Plot", fontsize = 25)
plt.show()
```

### **Line plot**

The first plotting technique to be used is the line plot. It uses a line to show the frequency of the data. It is the simplest and most used plot technique.

Example


```
x = [5,4,3,2,1]
y = [25,55,45,65,85]
y2 = [40,30,50,20,60]
y3 = [60,20,60,40,70]
y4 = [20,50,20,80,60]
plt.plot(x,y,'g',label='Maruthi',linewidth=3)
plt.plot(x,y2,'c',label='Jaguar',linewidth=3)
plt.plot(x,y3,'b',label='KIA',linewidth=3)
plt.plot(x,y4,'y',label='Ranger',linewidth=3)
plt.title('Car details in line plot', fontsize = 25)
plt.ylabel('Distance in kms')
plt.xlabel('Days')
plt.legend()
```


Output

### **Scatter plot**

A scatter plot displays the graph as a format of dots along the x and y axis of many variables.

Example


```
days = [5,4,3,2,1]
y1 = [25,55,45,65,85]
y2 = [40,30,50,20,60]
y3 = [60,20,60,40,70]
y4 = [20,50,20,80,60]
plt.scatter(days,y1,label='Jackfruit',color='r')
plt.scatter(days,y2,label='KIWI',color='b')
plt.scatter(days,y3,label='Gauva',color='y')
plt.scatter(days,y4,label='Mausie',color='g')
plt.xlabel('Days')
plt.ylabel('Price in Rs')
plt.title('Fruit details in Scatter Plot', fontsize = 25)
plt.legend()
```

### **Histogram Plot**

A histogram plot is a visual representation of numerical data that makes special use of rectangular blocks. The probability distribution may be estimated using a histogram. Space between the blocks cannot be included in this.

Example


```
days = [70,60,30,70,80,20,20,20,50,20,70,20,60,80,40,50,70,60,90,60]
bins = [0,10,20,40,50,60,70,80,90,100]
plt.hist(days, bins, histtype='stepfilled',color='green', rwidth=0.88)
plt.xlabel('Distance in kms', fontsize = 15)
plt.ylabel('Kilometer count', fontsize = 15)
plt.title('Car details Histogram', fontsize = 25)
```

### **Area plot**

This plot type is used to display quantitative data. An area plot's boundary is created by a line, and the space between the lines and the axis is colored.

Example


```
days = [5,4,3,2,1]
Maruthi = [25,55,45,65,85]
Jaguar = [40,30,50,20,60]
KIA = [60,20,60,40,70]
Ranger = [20,50,20,80,60]
plt.plot([],[],color='r', label='Maruthi', linewidth=3)
plt.plot([],[],color='g', label='Jaguar', linewidth=3)
plt.plot([],[],color='y', label='KIA', linewidth=3)
plt.plot([],[],color='b', label='Ranger', linewidth=3)
plt.stackplot(days, Maruthi, Jaguar, KIA, Ranger, colors=['r','g','y','b'])
plt.xlabel('Days', fontsize = 15)
plt.ylabel('Distance in kms', fontsize = 15)
plt.title('Car details in area plot', fontsize = 25)
plt.legend()
```
