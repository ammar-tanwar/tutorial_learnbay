# Chapter 3


# Tableau Data Format

 


## 1. Overview

Formatting in any application is the final presentation of the output. Likewise, Tableau also provides a vast selection of formatting options.


## 2. Scope of the tutorial

This tutorial covers


●  Different types of tableau data formats.

 

When modifying the presentation of the output visuals, Tableau offers a relatively huge selection of formatting choices. Almost all components, including **_font, color, size, and layout, can be modified_**. **_Tables, axis labels, workbook themes, and other containers_** can all be formatted in addition to the information.

 

The ‘Format’ option, which offers the possibilities, is displayed in the diagram below. This lesson will discuss some of the most popular tableau formatting options.


 <img src="https://learnbay-wb.s3.ap-south-1.amazonaws.com/main/tutorial/tableau/TF+-+1.PNG"></img>




1. **The Axis Formatting**

You can quickly build a bar chart by inserting the measured profit into the Rows shelf and the dimension Sub-Category into the Columns shelf. Choose the vertical axis by clicking on it. After that, select Format with a right-click.


 <img src="https://learnbay-wb.s3.ap-south-1.amazonaws.com/main/tutorial/tableau/TF+-+2.PNG"></img>




2. **Modifying the font**

The Format bar, which is located on the left, will appear. Select the font drop-down. As seen in the screenshot below, select Arial as the font style and 8 as the font size.


 <img src="https://learnbay-wb.s3.ap-south-1.amazonaws.com/main/tutorial/tableau/TF+-+3.PNG"></img>




3. **Changing the Alignment and Shades**

As demonstrated in the below example (image), one can also alter the shading color and the direction of the values on the axis.


 <img src="https://learnbay-wb.s3.ap-south-1.amazonaws.com/main/tutorial/tableau/TF+-+4.PNG"></img>




4. **Borders Formatting**

Imagine something like a crosstab chart with regions in the rows and Sub-Category in the columns. By utilizing the formatting options, you could now modify the crosstab table's borders. Select Format utilizing a right-click on the crosstab chart.

The left pane displays the Format Borders. Select the options as indicated by the screenshot below.


 <img src="https://learnbay-wb.s3.ap-south-1.amazonaws.com/main/tutorial/tableau/TF+-+5.PNG"></img>




5. **Column Formatting**

Rename - To change the column's name, double-click on it.

Copy - Copy and paste the selected row or column into Excel.

Split - Divide the column into spaces.



6. **Sorting Columns**

By default, Tableau displays the fields in the exact sequence as the source data. The choice to sort fields can be utilized to arrange these fields. This is one of the most used features in tableau data format. 


## 3. Sorting Rows/Data



* Clicking on the smaller icon with the field name to sort the information in ascending or descending order.
* _Quick Sort:_ We can sort the graph in either ascending or descending order utilizing the quick sort button.
* _Sorting from Toolbar:_ We must select the relevant pill and press the sort button to sort the toolbar.
* _Sorting by Pills_: We have limited options because this is in advance.
