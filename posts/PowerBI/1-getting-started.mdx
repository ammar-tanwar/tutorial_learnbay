# Power BI Introduction and Installation

## 1. Overview

**Power BI** is a business intelligence and data presentation tool that creates informational dashboards and insights from data from several sources. The Power BI package, which provides a wide range of products, connections, and services, includes the Power BI Desktop, mobile Power BI apps, and the Power BI service, depending on SaaS accessible for multiple devices. These services are utilized by business clients to collect data and produce BI insights. This tutorial gives a strong knowledge of _ **Power BI introduction** _ and installation.

## 2. Scope of the tutorial

This tutorial covers

-  **Power BI Introduction**

-  **Power BI Installation**

Power BI is a software for corporate data analysis and data presentation that creates dynamic dashboards and reports from data that come from many sources.

The Power BI package, which provides a wide range of products, connections, and services, includes the Power BI Desktop, mobile Power BI apps, and the Power BI service, depending on SaaS accessible for multiple devices. Many such services are implemented by corporate clients to collect data and produce BI solutions.

Power BI Services (SaaS - Software as a Service) is utilized to print reports, whereas the Power BI desktop application is utilized to produce reports. The Power BI mobile app is utilized to browse dashboards and reports.

## 4. Power BI Architecture  

The following elements are part of Power BI:

-  **Power BI Desktop** - On the dataset, this is utilized to generate data visualizations and reports.

-  **Power BI Gateway** - By linking to any on-premises data sources, one can keep the data relevant using the Power BI on-premises gateway without having to relocate any data. One can make use of the already-made investments when querying huge databases.

-  **Power BI Mobile Application** - Users can access data from anywhere by utilizing Power BI mobile applications. There are Power BI apps for the Windows, iOS, and Android operating systems.

-  **Power BI Service** - For publishing Power BI reports and data visualizations, utilize this cloud-based service.

A wide variety of data sources are supported by Power BI. The available data connections are displayed when you click 'Get data.' One can link to various SQL databases, flat files, the Azure cloud, or even web-based services like Facebook, Google Analytics, and Salesforce products. Moreover, it includes an ODBC connection so that users can access unlisted ODBC data sources.

The Power BI data sources that are accessed are listed below.

- SQL Database

- Blank Query

- Flat Files

- Online Services

- OData Feed

- Azure Cloud platform

- Other data sources such as Exchange, Hadoop, or Active Directory

To gather information, users need to choose the 'Get data' feature on the Power BI desktop's home page. The most popular data sources are displayed first. After that, select 'More' to get a list of all available data sources.
