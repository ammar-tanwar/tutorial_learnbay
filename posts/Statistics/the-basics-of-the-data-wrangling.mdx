## 3. Data wrangling :- What is it?

To make sophisticated data sets better user-friendly and understandable, data wrangling would be the method of cleaning up inconsistencies and mixing numerous complex data sets. The huge volume of data is needed to be maintained and managed for analysis since the amount of data and data types available these days are expanding rapidly.

Data wrangling, generally described as data munging, is the approach of restructuring, converting, and preparing data from one "raw" form into another to improve its significance and usefulness for a range of future uses, particularly analytics.


## 4. Instances of fundamental data munging tools include:



1. One very basic automated data-wrangling tool is spreadsheets and Excel Power Query.
2. Another automated data-cleaning tool that requires programming knowledge is named OpenRefine.
3. Tabula is a tool useful for all kinds of data.
4. The data service known as Google DataPrep examines, cleanses, and processes data.
5. Another tool that cleans and converts data is known as Data Wrangler.


## 5. Examples For Data Wrangling 

In a range of use cases, data-wrangling methods are utilized. The most typical applications for data wrangling usually include:



* Merging different data sources that create an individual data set for analysis.
* Recognizing data gaps and empty cells, whereupon restoring or removing it.
* Removing unwanted or useless data.
* Recognizing significant outliers in data, demonstrating the reasons they are inaccurate, or deleting them to enable tasks easy.
